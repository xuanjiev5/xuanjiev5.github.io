背包问题初始化的研究

本文最后更新于：2 年前

本讲义是一篇本人做背包问题题目时的一些感悟，文章较长~

学背包问题的过程:

  • 一开始学背包问题时遇到的大多数的状态表示是：从前i个物品中选，且总体积不超过j的问题。
  • 慢慢地就有出现状态表示是：从前i个物品中选，且总体积恰好是j的问题。例如买书 ，求的是恰好是j的总方案数问题。
  • 同时还出现了状态表示是：从前i个物品中选，且总体积至少是j的问题。例如潜水员 ，求的是总体积至少是j的最小价值

可以观察到，他们的分析方法以及状态转移方程都是一样的，唯独是初始化有很大的不同

例子：给你一堆物品，每个物品有一定的体积，每个物品只能选一个，求总体积不超过m的方案数

从前i个物品中选，且总体积不超过j的总方案数，初始化是f[0][i] = 1, 0 <= i <= m，其余是0

求最大值最小值问题

例子：给你一堆物品，每个物品有一定的体积和对应的价值，每个物品只能选一个，求总体积不超过m的最大价值

1、从前i个物品中选，且总体积不超过j的最大价值，初始化是f[i,k] = 0，0 <= i <= n, 0 <= k <= m（只会求价值的最大值）

例子：给你一堆物品，每个物品有一定的体积和对应的价值，每个物品只能选一个，求总体积恰好是j的最小价值
