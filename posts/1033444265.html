二分

大家对二分的刻板印象就是，二分只能解决查找问题，在一段序列查找一个特定的数，其实不然。

二分其实可以解决一大类问题，而且二分在算法竞赛中也尤为重要，那么他能解决什么问题呢？

解决的是二段性问题，只要存在二段性那么就可以使用二分来解决。

那啥是二段性呀？

就是给定一个条件时，俩边呈现俩个状态，且只有这俩个状态。

举个例子：

  • 就用二分查找来说吧： 给定一个值供你去寻找，那么这个排好序的左边就是比这个数小，右边就是比它大，这样就整个序列就呈现了二段性。
  • 或让你在一个只有一个根的函数上寻找根，那么这个根的左右俩侧值相乘小于零，同侧大于零
  • 等等等等

二分的应用性真的很广泛，不得不说二分已经成为必须要掌握的一个算法了。

机器人正在玩一个古老的基于 DOS 的游戏。

游戏中有 N+1 座建筑——从 0 到 N 编号，从左到右排列。

编号为 0 的建筑高度为 0 个单位，编号为 i 的建筑高度为 H(i) 个单位。

起初，机器人在编号为 0 的建筑处。

每一步，它跳到下一个（右边）建筑。

假设机器人在第 k 个建筑，且它现在的能量值是 E，下一步它将跳到第 k+1 个建筑。

如果 H(k+1)>E，那么机器人就失去 H(k+1)−E 的能量值，否则它将得到 E−H(k+1) 的能量值。
